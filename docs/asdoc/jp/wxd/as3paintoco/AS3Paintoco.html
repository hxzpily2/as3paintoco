AS3Paintocoはペイントコを利用する際に、インスタンス化するクラスです。

  // create AS3Paintoco
  var paint:AS3Paintoco = new AS3Paintoco(container, 465, 465);
  paint.applyTool(new PenTool());
  paint.activate();

save()method 
public function save(fileName:String, mode:String = "png", quality:Number = 80):void

キャンバスをローカルファイルシステムにファイルを保存するためのダイアログボックスを開きます。

ファイルの形式はPNG、またはJPEG形式です。

Parameters
fileName:String — 保存時の初期ファイル名を指定します。
 
mode:String (default = "png") — PNG形式で保存したい場合は、pngを指定します。JPEG形式で保存したい場合は、jpgもしくはjpegと指定します。
 
quality:Number (default = 80) — modeをjpgした際に使用されるJPEG画像の画質を0.0～1.0で指定します。modeをpngに指定した場合は無視されます。
