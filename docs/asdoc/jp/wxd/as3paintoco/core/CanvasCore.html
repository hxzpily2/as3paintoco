Packagejp.wxd.as3paintoco.core
Classpublic class CanvasCore
InheritanceCanvasCore flash.events.EventDispatcher

CanvasCoreクラスは、as3 Drawingライブラリを使用する際にインスタンス化するクラスです。 画を描く際、キャンバスのコントローラの役割を担います。
